Open Source Application

This application is open source, meaning everyone can see the code for it. If you're a developer and would like to improve upon it, the repositories are located here:

  • Tenant Search Server (https://github.com/ryanwaite28/tenant-search-server)
  • Tenant Search Client (https://github.com/ryanwaite28/tenant-search-client)

Simply make a pull request with you proposed changes.


The project is being hosted on Heroku.

The goal is to make this a free and public service that benefits everyone who may need something like this.
